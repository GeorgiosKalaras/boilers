-> pre-processing data

-> encoder-decoder transformer
   (test on one category of one house that it works)

-> genetic algorithm
   (test using the transformer for one category of one house)

-> create one model for each category (for all houses)
   (find parameters of best model with genetic algorithm)
   (train on data of the category from all houses)

-> create model that predicts boiler modulation
   (it will give prediction using previous boiler modulation and the predictions for the other categories)
   (find parameters for best model using genetic algorithm)
   (train on data from all houses)
(test on data from 20% of houses that had been randomly selected and were not used for training)
(repeat a few times, selecting different 20% of houses for testing)
(select model that performs best on most houses, without having too bad performance for the 'worst' house)
